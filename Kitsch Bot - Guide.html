1 Getting Started with Kitsch Bot

Welcome to Kitsch Bot! This tutorial will teach you everything you need to know to create stunning announcement embeds, schedule posts, and manage your server like a pro.

What is Kitsch Bot?

Kitsch Bot is a premium Discord bot that helps moderators create beautiful, customizable announcement embeds. Unlike regular Discord messages, embeds can have:

  • Custom colors ‚Äì Make your posts stand out
  • Images & thumbnails ‚Äì Add visual appeal
  • Interactive buttons ‚Äì Link to websites or trigger actions
  • Scheduled posting ‚Äì Set it and forget it
  • Sticky functionality ‚Äì Keep important posts visible

Required Roles

To use Kitsch Bot commands, you need one of these roles:

  • Admin
  • Event Organizer
  • Mod
Don't have the right role? Contact your server administrator to get access!

Your First Commands

Kitsch Bot uses slash commands. Just type / in any channel and you'll see available commands.

/help
Shows a quick overview of all Kitsch Bot features.
/guide
Displays the full bot guide directly in Discord.

üñºÔ∏è Images

You can add two types of images:

Thumbnail

  • Small image in the top-right corner
  • Great for: logos, icons, small graphics
  • Recommended size: 80x80 to 400x400 pixels

Banner Image

  • Large image at the bottom of the embed
  • Great for: event banners, promotional graphics
  • Recommended size: 1200x630 pixels (like social media images)
Image URLs must be direct links! They should end in .png, .jpg, .gif, or .webp. Links from Google search results usually won't work.
Where to get image URLs:
  • Upload an image to Discord, right-click it, and select "Copy Link"
  • Use image hosting sites like Imgur
  • Use your server's uploaded images

5 Scheduling Posts for Later

Want to post at a specific time? Kitsch Bot can schedule your embeds!

How to Schedule

  1. Start with /embed create as normal
  2. Select your channel
  3. Click "üìÖ Schedule" instead of "Post Now"
  4. A modal will appear asking for:
Field Format Example
Date YYYY-MM-DD 2024-12-25
Time HH:MM AM/PM 09:00 PM
All times are in Asia/Manila timezone (PHT). Make sure to convert if you're in a different timezone!

After Scheduling

The bot will confirm your scheduled post and show you when it will be posted. The embed will automatically appear at the scheduled time!

Great for: Weekly announcements, event reminders, midnight launches, and time-sensitive content!

6 Editing Existing Embeds

Made a typo? Need to update information? You can edit any embed that Kitsch Bot has posted!

How to Get the Message ID

  1. Enable Developer Mode in Discord:
    User Settings ‚Üí Advanced ‚Üí Developer Mode ‚Üí ON
  2. Right-click the embed message you want to edit
  3. Click "Copy Message ID"
Can't see "Copy Message ID"? Make sure Developer Mode is enabled in your Discord settings!

Editing the Embed

/embed edit message_id:1234567890123456789
Replace the number with your actual message ID.

This will open the embed builder with all your existing content pre-filled. Make your changes and save!

7 Making Embeds "Sticky"

Sticky embeds stay at the bottom of the channel, always visible even as new messages are sent!

Perfect For:

  • Important rules or guidelines
  • Active event information
  • Welcome messages in high-traffic channels
  • Current promotions or announcements

How to Set a Sticky Embed

  1. Create an embed or find an existing one's Message ID
  2. Go to the channel where you want it sticky
  3. Run this command:
/embed sticky action:Set Sticky message_id:1234567890123456789

How to Remove a Sticky

/embed sticky action:Remove Sticky
Run this in the channel where the sticky is set.
Only one sticky embed per channel is allowed. Setting a new one will replace the old one.

8 Using the FAQ / Knowledge Base

Create an interactive FAQ that members can browse themselves!

Setting Up FAQs

1. Add FAQ Entries

/faq add category:General question:How do I verify?
A modal will appear where you enter the answer.

2. View All FAQs

/faq list
Shows all FAQs with their IDs (needed for deletion).

3. Delete an FAQ

/faq delete id:5
Replace 5 with the actual FAQ ID from /faq list.

4. Deploy the FAQ Menu

/faq deploy
Posts an interactive FAQ menu that members can use!

How Members Use It

  1. They see a dropdown menu with categories
  2. Selecting a category shows all questions in that category
  3. Selecting a question shows the answer
  4. "Back" buttons let them navigate easily
Good FAQ categories: General, Rules, Events, Verification, Roles, Partnerships
